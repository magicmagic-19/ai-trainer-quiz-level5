# ai-trainer-level5 — 理论刷题

> 人工智能训练师（五级/初级工）理论知识刷题工具

纯本地运行的网页应用，599 道练习题。配合 [实操练习](../ai-trainer-practice/) 使用。

## 特点

- **零依赖**：单个 HTML 文件，不需要数据库和注册
- **本地存储**：刷题记录、错题、收藏、笔记保存在浏览器 localStorage
- **错题复练**：自动收集错题，支持筛选回顾
- **收藏标记**：对重点题目添加收藏和笔记

## 快速开始

### macOS

双击 `start.command`，浏览器会自动打开 `http://127.0.0.1:5177/`

### 其他系统

```bash
python3 -m http.server 5177
```

然后浏览器打开 `http://127.0.0.1:5177/`

> ⚠️ 请使用固定地址打开（http://127.0.0.1:5177），不要直接双击 index.html。直接打开使用 file:// 协议，会导致 localStorage 存储路径不一致，刷题记录可能丢失。

## 记录迁移

如果你之前用 file:// 打开有过刷题记录，切换到固定地址后记录会消失。

**迁移步骤**：
1. 启动服务后，打开 `migrate-file-records.html`
2. 点击"迁移到固定网址"
3. 在导入页点击"合并导入"
