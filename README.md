# 人工智能训练师五级 · 理论刷题

> 599 道练习题，纯本地运行，打开即用，免注册免登录。

## 📥 下载

点击右上角绿色的 **Code** 按钮 → **Download ZIP**，下载后解压到任意文件夹。

## 🚀 启动（三步）

### Mac

1. 解压后，双击文件夹里的 `start.command`
2. 如果提示"无法打开"，去 **系统设置 → 隐私与安全性** 点击「仍要打开」
3. 浏览器会自动打开刷题页面 ✅

> **如果找不到 Python 3：**
>
> 打开终端（启动台搜索「终端」），输入以下命令并回车：
> ```
> xcode-select --install
> ```
> 弹出窗口点「安装」，完成后重新双击 `start.command` 即可

### Windows

1. 解压后，双击文件夹里的 `start.bat`
2. 浏览器会自动打开刷题页面 ✅

> **如果没反应或提示找不到 Python：**
>
> 1. 打开 https://www.python.org/downloads/
> 2. 点击黄色大按钮下载安装包
> 3. 双击安装包，**务必勾选 ☑「Add Python to PATH」**
> 4. 点击 Install Now 完成安装
> 5. 重新双击 `start.bat` 即可

> ⚠️ 每次刷题都要用 `http://127.0.0.1:5177/` 这个地址打开，不要直接双击 index.html，否则刷题记录会丢失。

## ✨ 功能

- **错题本**：答错的题自动收集，随时回顾
- **收藏 + 笔记**：重点题目加收藏，写自己的记忆点
- **复刷模式**：二轮、三轮针对性巩固
- **题型筛选**：单选 / 多选 / 判断
- **模块切换**：按 Day 1~5 分模块练习
- **学习记录导出**：下载刷题数据随时查看进度
- **零依赖**：不需要安装任何软件，有浏览器就能用

## 🔄 记录迁移

如果你之前直接用 file:// 打开过 index.html，切换到固定地址后旧记录会看不到。

**找回步骤**：
1. 先启动服务（双击 `start.command`）
2. 双击 `migrate-file-records.html`
3. 点击「迁移到固定网址」→「合并导入」
