# 人工智能训练师五级 · 理论刷题

> 599 道练习题，纯本地运行，打开即用，免注册免登录。

---

## 📥 第一步：下载

点击右上角绿色的 **Code** 按钮 → **Download ZIP**

![下载步骤](https://docs.github.com/assets/cb-20363/images/help/repository/code-button.png)

下载完成后，**右键解压**到任意文件夹（桌面也行）。

---

## 🚀 第二步：启动

解压后打开文件夹，你会看到这些文件：

- `start.bat`（Windows 双击这个）
- `start.command`（Mac 双击这个）
- `index.html`
- 其他文件不用管

### 🪟 Windows 用户

> ⚠️ **必须先装 Python 3！** 没装的话往下看「第三步」。

1. 双击 `start.bat`
2. 弹出一个 **蓝色窗口**（PowerShell），显示启动信息
3. 浏览器自动打开刷题页面 ✅

### 🍎 Mac 用户

1. 双击 `start.command`
2. 如果提示"无法打开"→ **系统设置 → 隐私与安全性 → 仍要打开**
3. 浏览器自动打开刷题页面 ✅

> ⚠️ 每次刷题都通过双击 `start.bat`/`start.command` 进入，不要直接双击 `index.html`，否则刷题记录会丢失。

---

## 🐍 第三步：安装 Python 3（没有的话）

启动时会自动检测。如果提示「未检测到 Python」，按下面步骤安装：

### Windows 安装 Python 3

1. 打开浏览器，访问：**https://www.python.org/downloads/**
2. 点击页面正中间那个**黄色大按钮**（Download Python 3.x.x）
3. 下载完成后，**双击**安装包（在下载文件夹里）
4. ⚠️ **关键一步！** 安装界面底部有一行字：
   ```
   ☐ Add Python to PATH
   ```
   **务必点一下前面的方框，打上对钩！**（如果不勾，后面用不了）
5. 点击 **Install Now**，等进度条跑完
6. 安装完成后，关掉安装窗口，**重新双击 `start.bat`**

### Mac 安装 Python 3

1. 打开**终端**（启动台搜索「终端」或 Terminal）
2. 复制粘贴下面这行命令，按回车：
   ```
   xcode-select --install
   ```
3. 弹出窗口点「安装」，等完成
4. 重新双击 `start.command`

## ❓ 常见问题

**Q：双击 start.bat 闪一下就没了？**
A：PowerShell 可能不可用，电脑太老的话打开 cmd，手动输入：
```
cd 下载文件夹路径
python -m http.server 5177 --bind 127.0.0.1
```
然后浏览器打开 `http://127.0.0.1:5177/`

**Q：浏览器显示"无法访问此网站"？**
A：检查黑色/蓝色窗口是否还开着。关掉窗口 = 服务停止，重新双击启动即可。

**Q：能不能在手机/平板上刷？**
A：暂不支持，需要电脑。
