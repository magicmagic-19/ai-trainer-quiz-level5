# 人工智能训练师5级刷题网页

以后不要直接双击 `index.html` 打开。

请双击 `start.command`，它会用固定地址打开网页：

```text
http://127.0.0.1:5177/
```

这样浏览器会一直把刷题记录、错题、收藏、笔记保存到同一个地方。

如果你之前是用 `file://` 打开的，旧记录还在浏览器的另一个存储空间里。第一次切换到固定地址时，可能看不到旧记录。最简单的做法是：旧页面先不要清数据，后面需要的话再做一次记录迁移。

## 迁移旧记录

1. 双击 `start.command`，保持服务窗口不要关。
2. 双击 `migrate-file-records.html`。
3. 在打开的页面里点击“迁移到固定网址”。
4. 在导入页里点击“合并导入”，再打开刷题页。
