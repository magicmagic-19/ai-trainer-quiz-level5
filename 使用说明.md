# 使用说明

## 启动

**Mac：** 双击 `start.command`

**Windows：** 双击 `start.bat`

浏览器会自动打开固定地址：

```
http://127.0.0.1:5177/
```

请始终使用这个地址打开，保持刷题记录不丢失。

> ⚠️ 不要直接双击 `index.html`。直接打开使用 file:// 协议，会导致 localStorage 存储路径不一致。

---

## 安装 Python 3（如果没有）

### Windows

1. 打开 https://www.python.org/downloads/
2. 点击黄色大按钮下载
3. 双击安装包
4. ⚠️ **最重要的一步：勾选「Add Python to PATH」**
5. 点击 Install Now，等安装完成
6. 重新双击 `start.bat`

### Mac

1. 打开终端（启动台搜索「终端」）
2. 输入 `xcode-select --install` 回车
3. 弹出窗口点「安装」
4. 重新双击 `start.command`

---

## 迁移旧记录（file:// → http://）

如果你之前用 file:// 打开过，记录在另一个存储空间里。迁移步骤：

1. 双击 `start.command`，保持服务窗口不要关
2. 双击 `migrate-file-records.html`
3. 点击「迁移到固定网址」
4. 在导入页点击「合并导入」

完成后再打开刷题页即可看到旧记录。

## 常见问题

**Q：Mac 提示"无法打开 start.command"？**
A：去 **系统设置 → 隐私与安全性**，拉到最下面，点击「仍要打开」。

**Q：Windows 双击 start.bat 闪退？**
A：打开 cmd（Win+R 输入 cmd 回车），手动进入下载文件夹，运行：
```
python -m http.server 5177 --bind 127.0.0.1
```
浏览器打开 http://127.0.0.1:5177/

**Q：浏览器打不开 http://127.0.0.1:5177/？**
A：检查启动窗口是不是被关了。关掉窗口 = 服务停掉 = 打不开。重新双击启动即可。
